# Flipflops

Flipflops are Circuits that maintain a certain state unless and until directed by the input for changing that state<br>

Types of Flipflops:
<ul>
<li>SR Flipflops</li>
<li>JK Flipflops</li>
<li>D Flipflops</li>
<li>T Flipflops</li>
</ul>
